📊 数据处理与分析

处理表格数据、进行统计计算的核心工具。

1
pandas
数据分析的"瑞士军刀"
处理表格数据的首选库。读取 CSV、Excel、JSON 文件，进行筛选、分组、统计、合并——几乎所有数据分析任务都离不开它。
import pandas as pd df = pd.read_csv("news_data.csv") # 筛选阅读量 > 10万的新闻 hot = df[df["views"] > 100000] # 按来源分组统计 df.groupby("source")["views"].mean()
CSV/Excel读写 数据清洗 pip install pandas
2
numpy
高性能数值计算引擎
pandas 的底层依赖。提供高效的多维数组运算，是科学计算的基石。当你需要矩阵运算或大规模数值处理时必不可少。
import numpy as np arr = np.array([85, 92, 78, 95, 88]) print(f"平均分: {np.mean(arr):.1f}") print(f"标准差: {np.std(arr):.1f}") # 生成 0-1 随机数矩阵 matrix = np.random.rand(3, 4)
数组运算 统计计算 pip install numpy
3
openpyxl
Excel 文件读写专家
当 pandas 读写 .xlsx 文件时，底层就靠它。也可以直接操作 Excel 工作表，设置格式、合并单元格、插入图表。
from openpyxl import load_workbook wb = load_workbook("report.xlsx") ws = wb.active for row in ws.iter_rows(min_row=2, values_only=True): print(row[0], row[1]) # 标题, 来源 wb.save("report_updated.xlsx")
Excel读写 格式设置 pip install openpyxl

📈 数据可视化

把数据变成直观、有说服力的图表。

4
matplotlib
Python 绘图的"老大哥"
最经典的绑图库，支持折线图、柱状图、散点图、饼图等几乎所有图表类型。学术论文和报告中最常用。
import matplotlib.pyplot as plt x = ["科技", "娱乐", "体育", "财经"] y = [350, 280, 190, 220] plt.bar(x, y, color="#10b981") plt.title("新闻分类统计") plt.savefig("chart.png", dpi=150) plt.show()
学术图表 高度可定制 pip install matplotlib
5
wordcloud
一行代码生成词云
将文本中的高频词汇以视觉化方式呈现。字越大代表出现频率越高，非常适合展示舆情分析结果。
from wordcloud import WordCloud text = "人工智能 大数据 机器学习 深度学习 ..." wc = WordCloud( font_path="simhei.ttf", # 中文字体 width=800, height=400 ).generate(text) wc.to_file("wordcloud.png")
词云图 舆情可视化 ⚠️ 中文需指定字体
6
pyecharts
交互式图表神器
生成可交互的 HTML 图表（鼠标悬停、缩放、筛选），视觉效果比 matplotlib 更现代，非常适合做数据新闻展示。
from pyecharts.charts import Bar from pyecharts import options as opts bar = (Bar() .add_xaxis(["科技","娱乐","体育"]) .add_yaxis("新闻数量", [350, 280, 190]) .set_global_opts(title_opts=opts.TitleOpts( title="新闻分类统计"))) bar.render("chart.html")
交互式图表 数据新闻 pip install pyecharts

🕷️ 网络爬虫

从互联网上采集数据的利器。

7
requests
HTTP 请求的"人话版"
发送 HTTP 请求的标准库。GET 获取网页、POST 提交表单——一切网络数据采集的起点。语法极其简洁。
import requests url = "https://news.example.com" resp = requests.get(url, timeout=10) resp.encoding = "utf-8" print(resp.status_code) # 200 print(resp.text[:200]) # 网页前200字符
网页请求 API调用 pip install requests
8
BeautifulSoup
HTML 解析的"手术刀"
拿到网页源码后，用它精准提取你需要的内容——标题、链接、正文。配合 requests 使用，是爬虫的黄金搭档。
from bs4 import BeautifulSoup soup = BeautifulSoup(html, "html.parser") # 提取所有新闻标题 titles = soup.find_all("h2", class_="title") for t in titles: print(t.get_text(strip=True))
HTML解析 数据提取 pip install beautifulsoup4
9
selenium
浏览器自动化"司机"
当网站需要 JavaScript 渲染或登录才能看到内容时，selenium 能模拟真人操作浏览器——点击、滚动、输入，全都行。
from selenium import webdriver from selenium.webdriver.common.by import By driver = webdriver.Chrome() driver.get("https://weibo.com") # 等待并点击元素 btn = driver.find_element(By.CSS_SELECTOR, ".btn") btn.click() driver.quit()
动态页面 模拟登录 ⚠️ 需下载浏览器驱动

📝 文本分析

中文分词、情感分析、关键词提取。

10
jieba
中文分词的"国民工具"
把一段中文文本切分成一个个词语。支持精确模式、全模式、搜索引擎模式，还能自定义词典加入专业术语。
import jieba text = "汕头大学新闻传播学院开设基础编程课程" words = jieba.lcut(text) print(words) # ['汕头大学', '新闻', '传播', '学院', '开设', # '基础', '编程', '课程']
中文分词 关键词提取 pip install jieba
11
re (正则表达式)
文本匹配的"万能钥匙"
Python 内置库，无需安装。用模式匹配提取电话号码、邮箱、URL 等结构化信息，数据清洗必备技能。
import re text = "联系方式：13800138000，邮箱 test@stu.edu.cn" phone = re.findall(r"1[3-9]\d{9}", text) email = re.findall(r"\S+@\S+\.\S+", text) print(f"电话: {phone}") # ['13800138000'] print(f"邮箱: {email}") # ['test@stu.edu.cn']
模式匹配 数据清洗 内置库，无需安装
12
collections
词频统计的"快捷方式"
Python 内置库。其中 Counter 类专门用于统计元素频率，一行代码搞定词频排序，是文本分析的好帮手。
from collections import Counter words = ["人工智能","大数据","人工智能","机器学习", "大数据","人工智能","深度学习"] freq = Counter(words) print(freq.most_common(3)) # [('人工智能', 3), ('大数据', 2), ('机器学习', 1)]
词频统计 计数排序 内置库，无需安装
