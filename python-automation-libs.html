🤖 Internet Automation

9个Python 互联网自动化库

让你的代码像长了手一样——自动填表、截图、发消息、收邮件，甚至绕过API限制爬取数据。

💡
一句话理解：requests 和 selenium 是互联网自动化的"公交车"——能到，但慢、挤、费力。今天介绍的 9 个库，是为特定任务量身定做的"专车"——填表有专车、截图有专车、收邮件也有专车，上车即走，一行代码解决一个痛点。

🧰 九大神器

每一款都能让你的代码自动完成原本需要手动操作的任务。

1
MechanicalSoup
会填表的"乖浏览器"
就像一个听话的实习生，你告诉他"打开登录页，填用户名密码，点提交"，他就照做，不会多开一个标签页。
import mechanicalsoup browser = mechanicalsoup.StatefulBrowser() browser.open("https://github.com/login") browser.select_form('form[action="/session"]') browser["login"] = "your_username" browser["password"] = "your_password" browser.submit_selected() print(browser.get_url())
表单提交 简单登录 ⚠️ 不支持JS渲染
2
Pyppeteer
头号Chrome手术刀
Selenium像是开着卡车冲进商场，Pyppeteer则是派一个隐形人进去精准取货。首次运行会自动下载Chromium。
import asyncio from pyppeteer import launch async def screenshot(): browser = await launch(headless=True) page = await browser.newPage() await page.goto("https://news.ycombinator.com") await page.screenshot({"path": "hn.png"}) await browser.close() asyncio.run(screenshot())
JS渲染页面 截图自动化 ⚠️ 内存~200MB
3
Telethon
Telegram的"神级遥控器"
别人用机器人（Bot）是点外卖，你用Telethon是直接接管了整个厨房。国内可参考 wechatpy 或飞书SDK。
from telethon import TelegramClient api_id = 12345 api_hash = "your_api_hash" client = TelegramClient("anon", api_id, api_hash) async def main(): await client.start() async for msg in client.iter_messages("me"): print(msg.sender_id, msg.text) client.loop.run_until_complete(main())
消息自动化 数据存档 ⚠️ 需API Key
4
huey
轻量级定时任务"自动巡航"
你设定好"每隔5分钟检查一次网站状态"，它就像闹钟一样准时执行。比Celery轻量得多，但同样支持分布式。
from huey import RedisHuey, crontab import requests huey = RedisHuey('my_app') @huey.periodic_task(crontab(minute="*/5")) def ping_site(): r = requests.get("https://example.com") print(f"Pinged: {r.status_code}") # 命令行：huey_consumer.py my_app.huey
定时任务 监控巡检 国内替代：函数计算FC
5
imbox
把邮箱变成数据库
你每天手动下载附件，它帮你自动归类、解析、存盘。适合自动下载发票、提取验证码、监控客服邮件。
from imbox import Imbox with Imbox("imap.gmail.com", username="me@gmail.com", password="mypassword", ssl=True) as imbox: for uid, msg in imbox.messages(unread=True): print(msg.subject, msg.sent_from)
邮件自动化 附件下载 ⚠️ 国内邮箱需授权码
6
scdl
SoundCloud自动化下载器
像一个DJ助手，你告诉它"我要这位艺术家的所有作品"，它就去整理成MP3文件。适合构建音频数据集。
import subprocess subprocess.run([ "scdl", "-l", "https://soundcloud.com/artist/track", "-p", "./downloads" ]) # 执行后下载音频到指定目录
音频采集 语音预处理 ⚠️ 注意版权
7
Twint
绕过Twitter API的"挖掘机"
别人排队领号（申请API Key），你直接从后门进去翻数据。无API限制，但易被平台风控，需控制频率。
import twint c = twint.Config() c.Search = "python" c.Limit = 10 c.Store_csv = True c.Output = "tweets.csv" twint.run.Search(c) # 生成tweets.csv
社交数据采集 ⚠️ 已不可用，仅作历史参考 国内参考：weibo-scraper
8
Playwright
跨浏览器自动化"三栖特工"
Selenium是普通司机，Playwright是特技车手——还能同时开三辆车（Chromium、Firefox、WebKit）。比Selenium快约30%。
from playwright.sync_api import sync_playwright with sync_playwright() as p: browser = p.firefox.launch(headless=False) page = browser.new_page() page.goto("https://example.com") print(page.title()) browser.close()
UI自动化测试 录制生成代码 跨浏览器
9
youtube-search-python
绕过YouTube API的"白嫖搜索"
别人求着Google给API Key，你直接调用这个库像普通人搜索一样获取结果。适合构建教程推荐机器人。
from youtubesearchpython import VideosSearch videos = VideosSearch("python自动化", limit=3) results = videos.result() for video in results['result']: print(video['title'], video['link'])
视频搜索 数据采集 ⚠️ 非官方API

🇨🇳 Localization Tips

国内互联网自动化 "三板斧"

📱 微信生态

itchat（已失效）、wxauto（Windows版）、企业微信API——国内即时通讯自动化的主要入口。

☁️ 云服务替代

用函数计算（FC）替代huey，用OSS存储替代本地文件——免运维、按量计费。

🛡️ 反爬对抗

国内网站有更严格的风控，建议配合 pyppeteer-stealth 或 playwright-stealth 插件，并配置动态代理池。
